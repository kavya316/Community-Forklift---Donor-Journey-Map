# Community-Forklift---Donor-Journey-Map
Community Forklift Capstone Project (2025)

Information Science - 490 (INST490)

This repository contains all materials and deliverables for our University of Maryland capstone project with Community Forklift, a nonprofit organization focused on diverting waste and distributing reusable materials to local residents. 
Project Overview

Community Forklift’s mission is to reduce waste, promote sustainability, and make building materials accessible to everyone. Our team focused on understanding their operational workflows, donor interactions, and material intake processes. We developed a comprehensive donor journey map and actionable recommendations for improving efficiency and donor engagement.

Deliverables

The repository contains the following files and folders:

Community Forklift - Material Donation Journey Mapping - Final Paper.pdf – Final written report summarizing the project, findings, and recommendations.

Requirements Interview and Document.pdf – Document summarizing stakeholder interviews and requirements.

Project Plan (Part 1).pdf – Initial project plan outlining goals and timeline.

Project Plan (Part 2).pdf – Updated project plan with revised milestones and deliverables.

MID SEMESTER Presentation - Community Forklift – Mid-semester presentation slides.

FINAL Presentation - Community Forklift (12_11) – Final project presentation slides.

Journey Map IMG – Image of the donor and material journey map.

Community Forklift - Journey Map.jam – Miro/Figma file used to create the journey map.

2025 Community Forklift Customer Survey – Survey data provided by Community Forklift.

DonorTracker folder – Python scripts used for data analysis.

Data folder – Graphs and visualizations created from donor and pickup data.

CF Onsite Pictures – Photos from the warehouse visit and onsite observations.

Key Features

Comprehensive donor lifecycle journey map visualizing pain points and operational challenges.

Data analysis of donor pickup requests, including repeat donors and donation trends.
